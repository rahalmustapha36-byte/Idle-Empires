## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rahalmustapha36-byte/Idle-Empires.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Idle-Empires
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```
